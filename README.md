## How to Run

### Method 1: Python Server

1. Open Terminal
2. Navigate to the project folder:
   ```bash
   cd /Users/anshul/Desktop/FurnitureStore
   ```
3. Start the server:
   ```bash
   python3 -m http.server 8001
   ```
4. Open your browser and go to:
   ```
   http://localhost:8001
   ```

### Method 2: Open Directly

Simply double-click `index.html` in Finder to open it in your browser.
